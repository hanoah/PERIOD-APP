# Icons

## In-app icons

Nav, period button, history, doctor, stats, and onboarding use **Pepicons Pop** SVGs from [js/icons.js](../js/icons.js). Mode-specific custom SVG sets (alien/, girly/, neutral/) remain an optional future enhancement per PLAN.md.

## PWA install icons

`icon-192.png` and `icon-512.png` are used for "Add to Home Screen" on iOS and Android. They appear in the manifest and install prompts.
